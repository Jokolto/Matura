# Top-Down Action Shooter mit Lernenden Gegnern

Ein Top-down Action-Shooter, bei dem Gegner in Wellen erscheinen und während des Spiels lernen. Jede neue Welle wird aus den erfolgreichsten Gegnern der letzten Welle gebildet (Machine Learning Algorithmus). Der Spieler kann sich bewegen, schießen und ausweichen. Nach einer bestimmten Anzahl von Wellen erhält der Spieler Power-ups. Das Ziel des Spiels ist es, möglichst viele Wellen zu überleben.

## Features
- **Lernende Gegner:** Gegner passen sich an und werden mit jeder Welle stärker.
- **Power-ups:** Spieler erhalten nach bestimmten Wellen Power-ups.
- **Actionreiches Gameplay:** Bewegen, schießen und ausweichen, um zu überleben.

## Installation

1. **Repository klonen:**  
   Klone dieses Repository auf dein Gerät:
   ```bash
   git clone <repository-url>
   ```

2. **Godot installieren:**  
   Da es derzeit keine ausführbare Datei für das Spiel gibt, muss Godot installiert werden.  
   [Godot hier herunterladen](https://godotengine.org/download)

3. **Projekt öffnen:**  
   Öffne den Quellordner dieses Repositories als Godot-Projekt.

4. **Spiel starten:**  
   Starte das Projekt mit dem **Run-Button** oder drücke `F5`.

## Ziel des Spiels
Überlebe so viele Wellen wie möglich, während die Gegner immer intelligenter werden!
